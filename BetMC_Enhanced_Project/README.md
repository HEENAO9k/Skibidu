## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   pip install -r requirements.txt
   ```

2. Start the server:
   ```bash
   npm start
   # หรือ
   node server.js
   ```

3. Open browser: http://localhost:5000
